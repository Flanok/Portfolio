NASA project

front-end dev

Bingo is a package used for performing symbolic regression using genetic programming. Genetic programming is a type of machine learning that generates equations of arbitrary form to best fit data. While I interned at NASA, I was able to convert a large portion of the existing project from Python into C++ to increase optimization.

I also added non-blocking message passing interface to allow the project to be scalable. Lastly, I added more operations to allow for further modularization. Overall, the impact on the project was a 6x speed up.
